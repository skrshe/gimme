# Gimme (GIt MetagaME)
gm - A wrapper around the whole git meta game

`gm domain/user/repo`

go to [github.com/skrshe/gimme/wiki](options) for more info on internals

# Build Dependancies:
to build gimme:
- make
- gcc
- sudo
- git

to generate docs from the wiki:
- pandoc

# Installation
intall with:
```console
git clone https://github.com/skrshe/gimme
cd gimme/
make
sudo make install
```
remove with:
```console
sudo make uninstall
```
